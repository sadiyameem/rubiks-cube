# Rubik's Cube

# Description
- This is a 3D rubik's cube built with Three.Js. It is a 3x3 cube that can rotate, keyboard controlled and has a mouse control camera movement. This project includes a timer, start button, instructions, background changer and a zoom in/out effect.

# Features
- 3D Cube
- Keyboard controls
- Mouse control camera
- Timer
- Background changer
- Zooming effect (in/out)
- Start button
